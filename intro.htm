HARDWARE Y SOFTWARE
El hardware de la computadora, en términos simples, son los componentes físicos que un sistema de la computadora necesita para funcionar.
El software basico es un conjunto de programas que el sistema necesita para poder trabajar. Estos forman el Sistema Operativo del sistema. SOFTWARE DE APLICACIÓN. ... El Sistema Operativo permite al usuario manejar con alta facilidad todo lo referente a la instalación y uso de las redes de computadoras.
